## 📌 Project Overview

This is an **End-to-End Data Analysis Project** built on **Microsoft Power BI**, simulating real-world analytics for **PhonePe** — one of India's largest digital payment platforms.

The dashboard tracks and visualizes key financial services including **Insurance**, **Loans**, **Money Transfers**, and **Recharge & Bills**, along with a dedicated **Failed Payment Analysis** module. It is designed to reflect how data analysts work inside fintech companies — monitoring transactions, identifying failure patterns, and presenting insights to stakeholders.

This project is ideal for anyone looking to:
- Build a **portfolio-worthy Power BI project**
- Understand real-world **fintech data analytics workflows**
- Practice **end-to-end dashboard design** with interactivity and branding

## 🗂️ Dashboard Pages

| Page | Description |
|------|-------------|
| 🏠 **Homepage** | Overview of all services — total transaction amounts, failed payments summary, and date-range slicer |
| 🛡️ **Insurance** | Breakdown of insurance types — Term Life, Health, Car, etc. with monthly trends |
| 🏦 **Loans** | Analysis of loan categories — Auto Loans, Personal Loans — with amount vs. month charts |
| 💸 **Money Transfers** | UPI transfer trends, top transaction volumes, and payment status distribution |
| 📱 **Recharge & Bills** | Recharge and utility bill payment tracking with service-wise contribution charts |

---

## ❌ Failed Payment Analysis

A dedicated section on the **Homepage** deep-dives into failed transactions, analyzing:

- **Server Errors** — Backend failures affecting transaction completion
- **Insufficient Balance** — User-side payment failures
- **Wrong Payment Details** — Incorrect UPI IDs or account details
- **Other Technical Issues** — Network timeouts and gateway failures

> Visualized using a **Pie Chart** for quick failure-reason identification, enabling businesses to take corrective action.

- ✅ **5-Page Interactive Dashboard** — Full navigation between service pages via custom buttons

## 🚀 How to Use

1. **Clone this repository**
   ```bash
   git clone https://github.com/your-username/phonepe-powerbi-dashboard.git
   ```

2. **Download the dataset** from the link above and place it in the `/dataset` folder

3. **Open the `.pbix` file** in Power BI Desktop

4. **Refresh the data source** — update the file path to your local dataset location

5. **Explore all 5 pages** using the navigation buttons on the dashboard

> 💡 Or simply click **[View Live Dashboard](https://app.powerbi.com/view?r=eyJrIjoiYTY2YzYwOWEtMjE2MC00ZDMwLWE5YzAtMDA2NTJkOTM0OGJhIiwidCI6IjQ4ODdiYzNhLWNkZjgtNDY2ZS05MGI4LTljODAzMzM5MTYwNiJ9)** to explore it directly in your browser — no Power BI Desktop required!
